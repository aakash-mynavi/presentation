プロジェクトの目標

違法薬物使用者を予測する
機械学習モデルの開発

なぜ重要か

▸ 薬物依存は深刻な社会問題

▸ 早期発見 → 適切な介入 → 救える命

期待される効果

▸ 高リスク個人の早期特定

▸ 限られた資源の効果的配分

実現すること

✓ 客観的なリスク評価

✓ 早期介入の機会増加

✓ 資源の最適配分

✓ より多くの人を救う

データセット

地域リハビリテーションセンターで収集した1,877名のデータ。UCI Machine Learning Repository形式で整形。

人口統計学的特徴(3変数)

  • 年齢: 18-24, 25-34, 35-44, 45-54, 55-64, 65+
  • 性別: 男性・女性
  • 教育: 9段階(16歳以前の中退~博士号)

性格特性(7変数)

  • N: 神経症的傾向(Neuroticism)
  • E: 外向性(Extraversion)
  • O: 開放性(Openness)
  • A: 協調性(Agreeableness)
  • C: 誠実性(Conscientiousness)
  • I: 衝動性(Impulsiveness)
  • S: 感覚追求(Sensation Seeking)

目的変数

過去1年以内の違法薬物使用の有無を二値分類。使用者1,171名(62.4%)、非使用者706名(37.6%)。クラス不均衡を考慮した評価指標を採用。

統計手法

  • 探索的データ分析(EDA)
  • Pearson相関係数による相関分析
  • StandardScalerによる標準化処理
  • 層化サンプリング(80/20分割)
  • 主成分分析(PCA)の検討

目的: データの分布と特徴量間の関係を把握し、モデル性能を向上させる前処理を実施

機械学習アルゴリズム

  • Random Forest(アンサンブル学習)
  • Support Vector Machine(線形・RBFカーネル)
  • K-Nearest Neighbors(距離ベース)
  • Logistic Regression(線形分類器)

目的: 異なる学習原理を持つアルゴリズムを比較し、最適なモデルを選定

評価手法

  • 5分割層化交差検証(Stratified K-Fold)
  • GridSearchCVによるハイパーパラメータ最適化
  • 感度・特異度の最小値最大化
  • 混同行列による詳細分析

目的: 偽陽性・偽陰性のバランスを考慮し、実用的な予測性能を保証

使用ツール

  • Python 3.x
  • scikit-learn(機械学習)
  • pandas(データ処理)
  • seaborn / matplotlib(可視化)
  • imblearn(不均衡データ対応)

理由: 標準的なデータサイエンスライブラリで再現性と拡張性を確保

モデル 感度 特異度 精度
Random Forest 0.885 0.660 0.801

KNN (k=10) 0.821 0.645 0.755
Logistic Regression 0.872 0.617 0.777

結果の解釈

Random Forestが最高性能: 総合精度80.1%、感度88.5%(使用者の88.5%を正しく検出)、特異度66.0%(非使用者の66.0%を正しく検出)。SVMは感度が高いが特異度が低く、誤検知が多い。Random Forestは両者のバランスが最も良く、実用的なリスク評価ツールとして有効。n_estimators=480、max_depth=20で最適化。
